手动安装包

个人还是偏向手动安装第三方包
通过菜单条打开包路径
Preference –> Browse Packages
把第三方包放到该路径下面即可

我的常用包与配置

代码静态检查：SublimeLinter
代码提示： SublimeCodeIntel
生成MarkDown：Markdown Preview

修改配置文件的方法 Preference –> Packages Setting –> 选择你具体的包 –> Settings User

SublimeLinter 配置

{

    "sublimelinter": "load-save",
    "sublimelinter_popup_errors_on_save": true
}

SublimeCodeIntel 配置

{
    "codeintel_config": {
        "Python": {
            "env": {
                "PATH": "C:\\Python27"
            }
        }
    }
}

Markdown Preview 配置

{
    "browser": "C:\\Program Files\\Google\\Chrome\\Application\\chrome.exe",
    "enabled_parsers": ["markdown"],
    "enable_highlight": true
}

sublime 执行命令

ctrl + shift + p

Markdown 文件生成html文件
可以简单输入： mdp
